Problems figuring it out?

Select the avatar image DOM element, attach an event listener to the latter, that will be triggered when the user clicks on the image. Inside the event handler function, change the src attribute of the image for something else (ex: the image located at "image/avatar-bis.svg").

Problems figuring it out?

The "prompt" function is useful in this case. Create a button that opens a prompt when being clicked on. This prompt asks for the name of the user. When the user has given his name, change the text "Matt" in the "innerText" of the span tag (that has an id "name") with the given name and change its color to white using "element.style.color" property.

Problems figuring it out?

Just like you asked the user for its name, ask the latter to enter a color with "prompt". You can use that information to set the "style.backgroundColor" property, according to the color that was entered by the user,on the div element that has a class "description", inside the section element that has a class "presentation".

Problems figuring it out?

Change your previous code to get all elements with the class "pink-bg" and loop over them to set their "style.backgroundColor" property. Then, iterate over elements that have a class "pink-text" and change their "style.color" property

Problems figuring it out?
